# 功能范围说明

## 已完成功能（截至 2026-03-06）

1. 首页项目画廊
   - 3 列平铺卡片，支持搜索过滤。
   - 无项目时隐藏搜索栏并展示空状态。
   - 长按项目支持 `设置 / 排序 / 删除`。
2. 项目生命周期与快照
   - 一键创建本地项目目录和空白模板。
   - 手动快照最多 10 条，自动快照最多 10 条。
   - 支持快照列表加载与恢复。
3. 项目运行页
   - 全屏 `WKWebView` 加载本地入口文件。
   - 悬浮面板支持拖拽、边缘吸附、自动收起、再次展开。
   - 面板项：`刷新 / 聊天 / 文件 / 设置 / 退出`。
   - 新建项目未改动时退出会提示是否保留。
   - 内置 JS 错误桥接提示，避免静默失败。
4. 文件浏览与编辑
   - 文件浏览器支持目录递进与文件打开。
   - 文件内容页支持编辑和保存。
   - 集成 `Runestone`（可用时）并按后缀自动语法高亮。
5. Provider 管理
   - 支持 `OpenAI Compatible`、`Anthropic`、`Google Gemini`。
   - 每个 Provider 支持 `API Key` 与 `OAuth` 模式。
   - 支持自定义 Base URL、Model。
   - 表单字段有效时才允许提交。
6. 聊天改工程
   - 会话支持多线程（thread）持久化与切换。
   - 执行策略由模型路由：`direct_answer` / `single_pass` / `multi_task`。
   - 代码修改支持整文件覆盖与 `search/replace` 增量更新。
   - 使用 `json_schema` 约束输出，降低格式漂移。
   - 支持请求取消、阶段进度气泡、完成时长和请求级 token 用量显示。
   - 改动成功后自动快照，失败时保留已完成步骤结果。
7. Token Usage 统计
   - 设置页与项目页复用同一套 Dashboard。
   - 展示总输入/总输出 Token。
   - 固定 7 天图表，支持按周前后翻页。
   - 支持按 Provider/Model 维度查看日分布。
   - 长按柱状图可查看当日各分类明细。
8. 本地化与安全
   - Token Usage 页面文案已本地化（`en / zh-Hans / zh-Hant / zh-HK`）。
   - Provider 凭证存 `Keychain`，元数据存 `UserDefaults`。
   - 文件操作限制在项目目录，含路径安全校验。

## 下一阶段（路线图）

1. 聊天稳定性和可解释性增强
   - 更细的失败原因分层、重试策略可视化、结果回滚体验。
2. 运行调试能力增强
   - 更完整的运行日志与错误定位辅助。
3. Provider 生态增强
   - 更多模型参数控制与连接诊断。
4. 导出与分享
   - 项目打包导出与隐私检查。
