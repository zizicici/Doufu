# 功能范围说明

## 已完成功能（截至 2026-03-05）

1. 首页项目画廊
   - Safari 风格平铺卡片，约每行 3 列。
   - 右上角 `+` 新建项目，左上角进入设置。
   - 搜索栏支持项目名/描述过滤；无项目时自动隐藏。
2. 项目管理
   - 新建本地项目目录与空白模板文件。
   - 点击卡片进入项目运行页。
   - 长按项目弹菜单：`设置 / 排序 / 删除`。
   - 支持拖拽排序页面并持久化顺序。
   - 支持项目快照：
     - 用户手动保存快照（最多保留 10 条）。
     - 聊天成功修改后自动保存快照（最多保留 10 条）。
     - 支持在“载入快照”页面恢复到指定快照。
3. 项目运行页
   - 全屏 `WKWebView` 加载本地 `index.html`。
   - 可拖拽悬浮面板：`刷新 / 聊天 / 设置 / 退出`。
   - 新建后未修改项目退出时，提供“保存/不保存”确认。
   - 可捕获并提示页面 JS 运行错误（避免静默白屏）。
4. 项目设置
   - 项目级设置页（系统设置风格）支持：
     - 项目名称修改
     - 手动保存快照
     - 打开“载入快照”并恢复历史快照
5. Provider 设置
   - `Settings -> Manage Providers -> Add Provider` 流程完整。
   - 支持 `OpenAI / Compatible API`。
   - 认证方式：`API Key` 与 `OAuth`。
   - `Add Provider` 按钮仅在关键字段有效时可点击。
6. 聊天改工程（Codex）
   - 聊天请求携带当前项目文件上下文。
   - 模型返回 JSON 补丁后直接回写本地文件。
   - 小项目简单请求走单次快速路径，减少多次往返调用。
   - 支持两种改动方式：整文件覆盖与按片段 `search/replace`。
   - 请求启用结构化 `json_schema` 输出约束，降低格式漂移。
   - 支持流式解析、错误提示和完整调试日志。
   - 支持流式失败自动回退重试（`xhigh -> high`、`json_schema -> 普通文本`）。
   - 聊天成功有文件改动时自动创建快照，便于回退。
   - 会话记忆块会持久化到项目目录，重开聊天可延续上下文。
   - 输入区支持多行输入（`UITextView`）与动态高度。
   - 支持用户主动取消请求，取消会立即终止当前链路。
   - 多任务执行时会在每步改动后刷新文件上下文，降低后续步骤基于旧代码生成的概率。
   - 默认 `reasoning=high`，复杂任务可 `xhigh`。
7. 安全与存储
   - Provider 元数据保存在本地 `UserDefaults`。
   - API Key / Bearer Token 存储于 `Keychain`。
   - 项目文件访问限制在项目目录内。

## 下一阶段（路线图）

1. 聊天能力增强
   - 多轮稳定编辑、失败恢复、增量 diff 级别回滚。
   - 更细粒度的变更说明与冲突处理。
2. 项目运行增强
   - 控制台日志可视化、快速预览快照、错误定位。
3. Provider 能力增强
   - 更多 Provider 类型、连接状态检测、可用性诊断。
4. 分享与导出
   - 可控导出项目包，在不破坏沙盒隔离前提下支持分享。
