# 功能范围说明

## MVP 功能（当前阶段）

1. 项目入口管理
   提供“新建网页”和“打开已有网页”两个主入口。
2. 网页项目创建
   用户输入项目名和需求描述后，创建本地项目目录。
3. 本地网页运行
   支持在 App 内预览本地 `html + css + js`。
4. 基础项目管理
   查看项目列表、打开项目、删除项目（带确认）。
5. 沙盒隔离
   每个项目具备独立目录，限制越权访问。

## 后续阶段功能（路线图）

1. LLM 生成增强
   多轮修改、差异更新、失败重试和版本回滚。
2. 编辑与调试能力
   代码编辑、实时刷新、控制台日志。
3. 分享能力
   生成可分享包或导出方式（在不破坏本地安全前提下）。
4. 模板市场
   提供常用场景模板，如 ToDo、记账、打卡、表单。

## 非目标（MVP 暂不覆盖）

1. 云端多人实时协作。
2. 复杂后端托管能力。
3. 跨设备强同步（可在后续版本评估）。
