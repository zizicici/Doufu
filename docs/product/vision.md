# 产品愿景

## 一句话愿景

让普通用户无需传统编程背景，也能通过自然语言在本地快速创建、运行并迭代自己的小型 App（基于 `html + css + js`）。

## 核心目标

1. 降低创建门槛：用户只描述需求，就能得到可运行网页。
2. 保持本地可控：网页项目默认在本地创建、运行、编辑。
3. 支持后续分享：未来可以把本地项目分享给他人使用。
4. 安全优先：用户数据在本地沙盒隔离，项目之间互不影响。

## 目标用户

1. 想做效率工具但不会完整开发流程的普通用户。
2. 需要快速做原型验证的个人创作者。
3. 希望把小工具分享给朋友或团队的轻量开发者。

## 产品原则

1. `Local First`：核心体验在离线/本地也应可用。
2. `Prompt to Product`：从想法到可运行页面路径尽可能短。
3. `Safe by Default`：默认最小权限、默认隔离执行。
4. `Gradual Power`：新手可用，进阶用户可逐步控制细节。

## 北极星指标（建议）

1. 新用户首次创建成功率。
2. 从输入需求到首个可运行页面的平均时长。
3. 7 日内复访并继续编辑项目的比例。
